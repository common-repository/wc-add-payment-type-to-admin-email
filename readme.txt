== Description ==

WC Add Payment Type to Admin Email plugin adds a functionality to Woocommerce plugin that displays payment method used by customer in admin e-mail. This plugin requires Woocommerce.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `hide-related-products.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. You are done.
